# Project Context — Viewlytics

## Vision

Modern analytics intelligence platform for creators.

Focused initially on:
- Dominican creators
- podcasts
- streamers
- gaming creators
- digital media

---

## Main Objective

Build a scalable analytics SaaS capable of:
- tracking creator growth
- visualizing analytics
- detecting trends
- generating social-ready content

---

## Positioning

NOT a SocialBlade clone.

This platform should feel:
- premium
- modern
- minimal
- scalable

Inspired by:
- Stripe
- Linear
- Vercel
- Arc Browser

---

## Product Philosophy

Frontend is only the visualization layer.

The true value comes from:
- historical data
- analytics intelligence
- ranking systems
- insights
